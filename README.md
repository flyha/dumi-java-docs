# dumi-docs

A static site base on [dumi](https://d.umijs.org).

## Development

```bash
# install dependencies
$ npm install

# start dev server
$ npm start

# build docs
$ npm run build

#取消http代理
git config --global --unset http.proxy

#取消https代理
git config --global --unset https.proxy
```
